# WebStudio — Landing Page

Responsive single-page landing for a web development studio.

> Course project — [GoIT Full Stack Web Development](https://goit.global/) program.

## 🛠 Stack
- HTML5 (semantic markup, `<picture>`, `srcset`)
- CSS3 (Flexbox, Grid, media queries, transitions)
- JavaScript (vanilla) — mobile menu, modal window
- Google Fonts, SVG icon sprite

## ✨ Features
- Responsive layout: mobile / tablet / desktop
- Mobile menu that opens/closes on click (including clicking a nav link)
- Feedback modal with form validation
- Anchor links to Portfolio and Contacts sections
